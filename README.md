# Portfólio (Vue + Pinia)

Projeto de portfólio pessoal feito em Vue 3 com Pinia e Tailwind CSS.

## Rodar localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Rode o servidor de desenvolvimento:
   `npm run dev`
